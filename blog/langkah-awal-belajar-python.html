First Step With Python

Posted by Rifandani,  19 January 2020

Apa Itu Python?

Python, dinamai berdasarkan kelompok komedi Inggris Monty Python, adalah bahasa pemrograman yang ditafsirkan, interaktif, dan berorientasi objek. Fleksibilitasnya memungkinkannya melakukan banyak hal, baik besar maupun kecil.

Python dapat digunakan untuk menulis program sederhana, tetapi ia juga memiliki kekuatan penuh yang diperlukan untuk membuat solusi perusahaan berskala besar yang kompleks. Beberapa cara penggunaan Python meliputi:

  1. Pengembangan aplikasi grafis desktop, termasuk game
  2. Analisis data matematis dan ilmiah atau yang biasa disebut Data Science
  3. Web and internet development

Kehadiran Python di dunia pemrograman komputer dapat ditemukan di mana-mana. Sebagai contoh, Python digunakan di beberapa situs internet terbesar di dunia - seperti Reddit, Dropbox, dan Youtube.

Framework Web Python yang populer, Django, mendukung Instagram dan Pinterest. Perusahaan efek visual pemenang penghargaan LucasFilms, Industrial Light & Magic, menggunakan Python untuk membuat keajaiban efek visual mereka menjadi hidup.

Memulai Python

Python adalah perangkat lunak open-source gratis yang bekerja di Linux, Mac, Windows, dan berbagai platform lainnya (total ada 21). Python sudah terinstall pada Mac dan sebagian besar distribusi Linux, namun Anda mungkin perlu mengunduh versi terbarunya disini.

Untuk memeriksa versi Pythonmu, buka terminal dan jalankan command berikut:
python -V

Memilih Versi Python

Ini dapat membingungkan bagi programmer baru dan berpengalaman dalam memutuskan antara Python 2 atau 3. Untungnya, ada beberapa perbedaan sintaks antara keduanya dan Anda dapat menjalankan kedua versi di komputer Anda.

Saat ini tersedia 2 versi utama bahasa Python, yaitu versi 2.x dan versi 3.x
Lalu mana versi yang tepat untukku?
Well, that's up to you. Sebenarnya, ada beberapa perbedaan antara keduanya yang akan mempengaruhi Anda pada tahap awal ini, namun hal itu tidak masalah pada saat ini.

Secara umum, jika Anda baru mulai mempelajari Python, gunakan versi 3.x, karena ini adalah versi yang lebih baik karena ada fitur yang ditambahkan bersama dengan banyak perbaikan bug dan penyempurnaan.

Namun, versi 2.x memiliki lebih banyak support untuk library pihak ketiga. Jika Anda tahu Anda harus menggunakan perpustakaan yang belum support ke versi 3.x, maka mungkin yang terbaik adalah mulai dengan versi 2.x

Installing Python

Terlepas dari sistem operasi Anda, Anda dapat mengunduh Python dari Python Software Foundation (PSF). Ambil versi khusus untuk OS dan prosesor Anda (32 atau 64-bit).

  • Mac: Saya sangat merekomendasikan menggunakan Homebrew untuk menginstal dan mengelola berbagai versi Python. Lihat petunjuknya disini. Anda juga dapat mengunduh Python dari PSF.
  • Linux: Python sudah pre-installed dalam berbagai distro Linux. Pastikan untuk meng-upgrade ke versi terbaru menggunakan package manager, jika perlu.
  • Windows: Anda dapat mendownload Python secara langsung melalui PSF.

Coding Primitives

Pada bagian ini, kita akan membahas kode primitif bawaan dari Python, yaitu Variable, Built-In Data Types, Comments, Syntax, dan Semantics.

~ Variables

Variabel adalah wadah untuk data. Sintaks untuk mendeklarasikannya adalah: variable_name = variable_value. Meskipun Anda dapat memberi nama variabel apa pun yang Anda suka (kecuali untuk beberapa reserved keywords), Anda harus menggunakan skema penamaan yang masuk akal secara intuitif. Nama variabel harus memberikan beberapa indikasi tentang nilai-nilai yang diberikan padanya.

Sebagai contoh, banyak programmer baru, menggunakan nama variabel sederhana seperti x dan y. Sebaliknya, jika Anda memiliki dua angka, misalnya 1 dan 52, gunakan nama variabel num1 dan num2 alih-alih x dan y. Dengan begitu ketika orang lain membaca program Anda, mereka dapat menebak bahwa num1 dan num2 mungkin merupakan angka. Pikirkan orang lain ketika menulis program Anda. Pastikan kode yang anda ketik readable bagi pembacanya, termasuk anda sendiri.
